Design Notes:

  • Card List Style (Preferred): More compact, easier to scan many work orders, familiar from current schedule page, better for detailed information display
  • Move Functionality: "Move" button opens a dropdown menu showing all available WH (Wire Harness) locations. User selects destination location to move the work order.
  • Location Filtering: Dropdown only shows locations beginning with "WH" (Wire Harness work centers). Current location is highlighted in the dropdown.
  • Complete Schedule Option: Checkbox allows user to choose whether to complete the schedule entry (checked = complete, unchecked = leave open)
  • Move Behavior: User clicks "Move" → Dropdown appears with WH locations → User selects destination → API call with selected location_id and complete_schedule: true/false → Work order moves to selected location → Auto-refresh
  • User ID: Automatically pulled from logged-in user's session for audit trail in CETEC
